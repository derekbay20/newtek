== Description ==

Best free blog and magazine theme for writers who want minimal layouts with best customization options. This theme is simple, clean, SEO optimized with eCommerce support takes you to the next level to your blog, magazine and your ecommerce store. You can use any page builder to build multiple sections to your website with this theme. Check demo at https://demo.themecentury.com/wpthemes/slain/

== Installation ==

1. In your admin panel, go to Appearance > Themes > click the Add New button.
2. Type Slain on search bar and click on install button
3. Click Activate to use your new theme right away.
4. Go to Dashboard > Appearance > Customize and change settings
5. Go to Dashboard > Appearance > Widgets to manage sections

== Frequently Asked Questions ==

= Does the theme is translation ready? =

* Yes, this theme is translation ready

= Does this theme support RTL Language?

* Yes, this theme is developed for ltr and rtl both friendly.

= Does this theme support any plugins? =

* Yes, We have mainly tested with WooCommerce, Century ToolKit, Contact Form 7 but you can use any plugins and let us know if you have any problem, We will fix if there any issue occures.
